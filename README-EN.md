# Johari Window - PTSTAR Personal Branding

## 📋 Project Overview
The Johari Window is a psychological tool for improving self-awareness and interpersonal relationships.  
This web application was developed as part of the [PTSTAR Personal Branding Program](https://ptstar.netlify.app/johari-window.dc).

## 🎯 Key Features
- Self-assessment by selecting 5 out of 50 adjectives
- Collect and compare peer evaluations
- Visualized results divided into 4 quadrants
  - **Open Area**: Known to self and others
  - **Hidden Area**: Known to self but not to others
  - **Blind Spot**: Unknown to self but known to others
  - **Unknown Area**: Unknown to self and others

## 🚀 Getting Started
1. Clone the repository
```bash
git clone [repository-url]
```

2. Firebase Setup
- Create a Firebase project
- Enable Firestore database

3. Deploy
```bash
firebase deploy
```

## 💡 How to Use
1. Enter your name
2. Select 5 adjectives that describe you
3. Share the link for peer assessments
4. View your Johari Window results
